# Základy UX

Balík poznámek pro „core“ znalosti UX/HCI: historický kontext designu, kognitivní základy interakce, principy z psychologie relevantní pro UI a etika (výzkum, soukromí, GDPR, dark patterns).

## Doporučené pořadí učení (propojení kapitol)
1. `human-computer-interaction.md` (zákony, modely, chyby, heuristiky)
2. `designove-principy-psychologicky-vyzkum.md` (LoC, zátěž, zkreslení, rozhodování)
3. `etika-designu.md` (férovost, privacy-by-design, výzkumná etika, GDPR)
4. `historie-designu.md` (proč ty principy vznikly a jak se přelévají do digitálu)

## Literatura

- Midal, A. (2019). Design by Accident. Sternberg Press. ISBN: 9783956791437.
- Norman, D. (2013). The Design of Everyday Things: Revised and Expanded Edition. Basic Books, 2013. ISBN 9780465055715.
- Kahneman, D. (2011). *Thinking, Fast and Slow.*
- Tversky, A., Kahneman, D. (1979). *Prospect Theory: An Analysis of Decision under Risk.*
- Nielsen, J. (1994+). *Usability heuristics.*
- Shneiderman, B. *Eight Golden Rules of Interface Design.*
- Card, S. K., Moran, T. P., Newell, A. (1983). *The Psychology of Human-Computer Interaction.* (MHP, GOMS/KLM)
- Rogers, Y., Preece, J., Sharp, H. *Interaction Design.* (metody a rámce HCI)
- Thaler, R. H., Sunstein, C. R. (2008). *Nudge.*
- Nissenbaum, H. (2010). *Privacy in Context.* (contextual integrity)
- ISO 9241-11 / ISO 9241-110 / ISO 9241-210 (použitelnost, dialogové principy, human-centred design)
